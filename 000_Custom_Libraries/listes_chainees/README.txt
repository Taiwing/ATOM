La bibliothèque "data_list" permet l'inclusion d'un système de listes chaînées.

Ces listes agissent de façon similaire à des tableaux, à ceci près qu'elles sont bien plus souples.
Elles peuvent être agrandies, réduites et détruites à souhait. On peut enregistrer plusieurs types
de données dans une seule liste et/ou dans un seul élément de celle-ci.

La bibliothèque peut donc s'inclure dans tout programme nécessitant ce type de gestion des données. Il faudra cependant l'adapter en fonction du type données que l'on souhaite gérer.
