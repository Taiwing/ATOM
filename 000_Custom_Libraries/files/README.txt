"data_files" est une bibliothèque permettant de gérer des données via un système de file.

Le système de file repose sur un alogirithme "FIFO" ("First In First Out"), qui permet d'enfiler des données les unes après les autres, et d'accéder à la première donnée enfilée lors du défilement (losqu'on enlève des éléments de la file).

La bibliothèque peut-être utilisée dans tout type de programme, il faudra cependant l'adapter en fonction du type de données que l'on veut gérer.
